Graph Information

Here is some brief information about different types of graphs and their usages:

  • Bar Chart

    A bar chart is used to compare different categories of data. It shows the frequency or count of items in each category using rectangular bars.

  • Line Chart

    A line chart is used to show trends over time. It connects data points using straight lines, making it easy to see how values change over a period.

  • Scatter Plot

    A scatter plot is used to show the relationship between two variables. It displays data points as dots, which can help identify correlations or patterns.

  • Pie Chart

    A pie chart is used to show the proportion of different categories within a whole. Each slice represents a category's contribution to the total.
